For the extension to do its thing you will need an access token for the GitHub API.

  1. Go to https://github.com/settings/tokens
  2. Click "Generate new token".
  3. Enter a token description, e.g. "Issue Preview Extension".
  4. Check the "repo" checkbox.
  5. Click "Generate token".
  6. Copy the token and paste it in the textfield below.
  7. Click "Save".
